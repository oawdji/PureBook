# 产品需求文档（PRD）— PureBook 记账应用

## 1. 项目概述

PureBook 是一款面向个人的 Web 记账应用，以丰富的图表和直观的交互界面帮助用户记录日常收支、分类管理，并通过可视化的统计仪表盘展示财务状况，让个人财务管理变得简单而优雅。

## 2. 目标用户

| 角色 | 说明 | 使用场景 |
|------|------|----------|
| 个人用户 | 唯一使用者，管理自己的日常财务 | 记录每日消费、查看月度开销分布、了解收支趋势 |

> 当前为单用户版本，不涉及多用户、角色权限或共享账本。

## 3. 核心功能列表

### Must Have（必须实现）

- **收支记录 CRUD**：添加、编辑、删除、查看收支记录
- **分类管理**：预设分类（餐饮、交通、购物等），支持自定义分类
- **月度统计概览**：本月收入/支出汇总、分类占比、收支趋势

### Should Have（应该实现）

- **记录搜索与筛选**：按日期范围、分类、金额范围筛选记录
- **数据导出**：导出 CSV 格式的账单数据
- **账户管理**：支持多账户（现金、银行卡、信用卡等）

### Nice to Have（可选实现）

- **预算设定与提醒**：设定月度预算，超支时提醒
- **周期性账单**：自动记录固定收支（房租、工资等）
- **数据备份与恢复**：一键备份/恢复数据库

## 4. 功能详情（用户故事 + 验收标准）

### 功能：收支记录 CRUD

**用户故事**：作为个人用户，我希望能够添加、查看、编辑和删除每笔收支记录，以便准确追踪我的资金流动。

**验收标准**：
1. 用户可以创建一笔记录，包含：金额、类型（收入/支出）、分类、日期、备注（可选）
2. 用户可以编辑已创建的记录，所有字段可修改
3. 用户可以删除某条记录，删除前弹出确认提示
4. 记录列表按日期倒序排列，默认显示最近 30 天的记录
5. 列表支持分页加载，每页 20 条

### 功能：分类管理

**用户故事**：作为个人用户，我希望收支能按分类归类（如餐饮、交通、工资等），以便了解钱花在哪里、从哪里来。

**验收标准**：
1. 系统预设至少 8 个常用分类（餐饮、交通、购物、住房、娱乐、医疗、工资、其他）
2. 每个分类有对应的图标和颜色标识
3. 用户可以创建自定义分类，并选择图标和颜色
4. 用户可以编辑和删除自定义分类（预设分类不可删除）
5. 删除分类时，如果该分类下有记录，提示用户将这些记录迁移到其他分类

### 功能：月度统计概览

**用户故事**：作为个人用户，我希望看到每月的收支汇总和分类占比，以便了解月度财务状况和消费习惯。

**验收标准**：
1. 统计页面展示当月总收入、总支出、结余
2. 展示支出分类占比饼图
3. 展示近 6 个月的收支趋势柱状图
4. 支持切换查看不同月份的统计数据
5. 点击饼图中的分类，可钻取查看该分类下的记录明细

### 功能：记录搜索与筛选

**用户故事**：作为个人用户，我希望快速找到某笔特定记录，以便核对账目。

**验收标准**：
1. 支持按关键字搜索备注字段
2. 支持按日期范围筛选
3. 支持按分类筛选
4. 支持按金额范围筛选
5. 多个筛选条件可组合使用

### 功能：数据导出

**用户故事**：作为个人用户，我希望将账单数据导出为 CSV 文件，以便在 Excel 中做进一步分析或存档。

**验收标准**：
1. 支持导出指定日期范围内的记录为 CSV
2. CSV 文件包含所有记录字段（金额、类型、分类、日期、备注）
3. 文件名包含导出日期范围，如 `PureBook_2025-01_2025-06.csv`

### 功能：账户管理

**用户故事**：作为个人用户，我希望管理多个资金账户（现金、银行卡等），以便区分不同来源的收支。

**验收标准**：
1. 用户可以创建多个账户，每个账户有名称、类型、初始余额
2. 每条收支记录关联一个账户
3. 账户余额自动根据关联的收支记录更新
4. 支持账户间转账

## 5. 非功能需求

### 性能
- 页面首次加载时间 < 2 秒
- 记录列表分页查询响应时间 < 300ms
- 月度统计数据查询响应时间 < 500ms
- 支持本地缓存，减少重复请求

### 安全
- 单用户版本，使用简单的密码登录保护即可
- 所有用户数据通过 HTTPS 传输
- 密码使用 bcrypt 加密存储
- API 使用 JWT Token 鉴权，Token 有效期 7 天

### 兼容性
- 支持现代浏览器：Chrome 90+、Firefox 90+、Edge 90+
- 桌面端优先，推荐 1280px 以上屏幕宽度

### 可用性
- 界面语言为中文
- 金额输入支持小数点后两位
- 日期选择使用日期选择器，格式统一为 YYYY-MM-DD
- 所有删除操作有二次确认
- 网络异常时给出友好提示

### 视觉与交互设计
- **仪表盘首页**：采用卡片布局展示本月收支概览，关键数字用醒目的大字体 + 数字滚动动效，配以趋势迷你图
- **统计页**：图表为主、表格为辅，饼图和柱状图需有入场动画和平滑过渡
- **记录列表**：每条记录左侧显示分类图标+颜色标识，收入/支出用绿色/红色区分；鼠标悬停行高亮 + 微放缩
- **添加记录**：使用侧边抽屉面板或模态弹窗滑入，内部字段逐项渐显
- **空状态**：各页面在无数据时有精美的插画式空状态提示，而非白屏
- **过渡动画**：页面路由切换淡入淡出、弹窗/抽屉滑入滑出、列表项进出场 stagger 动画
- **按钮与操作反馈**：所有按钮 hover/active 有视觉反馈；点击操作有微交互（涟漪效果、加载旋转）
- **深色模式**：支持浅色/深色主题切换，切换过程有平滑颜色过渡

### 可维护性
- 使用 Docker 容器化部署，支持一键启动
- 数据库和缓存独立部署，支持备份恢复
- 前后端分离，可独立升级
